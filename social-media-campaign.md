## Part 9: Quick-Start Brand Templates (Canva/Figma Specs)

### Color Palette for All Graphics
- Background: `#0a0a0f` (deep dark)
- Card background: `#111118`
- Primary text: `#E8E6F0` (off-white)
- Secondary text: `#9895A8` (muted gray)
- Accent: `#c4956a` (copper)
- Accent hover: `#d4a57a` (light copper)
- Success green: `#7fb885`
- Error red: `#e05c5c`

### Fonts
- Headlines: Instrument Serif (or Georgia as fallback)
- Body: DM Sans (or system sans-serif)
- Code/Technical: JetBrains Mono

### Post Dimensions
- Feed post: 1080 x 1080px (square) or 1080 x 1350px (4:5 portrait — more screen real estate)
- Story: 1080 x 1920px
- Reel cover: 1080 x 1920px
- Carousel: 1080 x 1350px per slide

### Design Rules
- Always use dark backgrounds (never white/light)
- Copper accent for highlights, CTAs, emphasis
- Generous whitespace / padding (min 80px margins)
- Serif for headlines, sans for body
- No emojis in graphics (use sparingly in captions if at all)
- Grain texture overlay at 3-5% opacity for warmth
- Subtle border lines in `rgba(196, 149, 106, 0.12)` for separation

## Appendix: Caption Formula

Every post should follow this structure:

1. **Hook** (first line — this shows in preview): A bold statement, question, or statistic
2. **Problem** (2-3 sentences): Describe the pain point they're living with
3. **Bridge** (1 sentence): Connect their situation to the solution
4. **Solution** (2-3 sentences): What you build / how it works
5. **CTA** (last line): Clear next step — link in bio, DM, or "see the full case study"

Keep it under 300 words. Use line breaks liberally. No walls of text.

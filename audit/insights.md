# audit: /insights (InsightsPage)

## verdict: clean blog index. needs more content to feel lived-in.

### what works
- clean card layout for posts
- category + read time + date metadata is well placed
- hover reveals "read more" link — subtle and tasteful
- CTA "building something original?" is on-brand

### issues

1. **hero subhead has the 6-word audience list again** — "artists, creatives, changemakers, practitioners, facilitators, and organizations"
   - **rewrite**: "Notes on building infrastructure for original work — the patterns, the problems, and why most off-the-shelf tools fail."

2. **no category filter** — the work page has filter pills by audience type. insights should have them too (or at least by category: infrastructure, partnership, etc.)

3. **depends on blogPosts data** — need to verify how many posts exist and whether they have meaningful content or are placeholder

4. **no newsletter signup** — a blog page without email capture is a missed opportunity. add a simple "get notified when I write" email field below the posts list.

5. **CTA is generic** — same "Start a Conversation" button used everywhere
   - **rewrite**: "Have a question about something I wrote? Tell me."

### structural fixes
- add category filter pills
- add newsletter email capture
- trim hero subhead

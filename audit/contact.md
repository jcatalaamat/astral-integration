# audit: /contact (ContactPage)

## verdict: clean and functional. minor improvements.

### what works
- form + sidebar layout is well structured
- calendly card with accent border draws attention
- "how did you find me?" referral field is smart attribution
- multiple contact options (form, calendly, email, linkedin, github)
- "I take on a small number of partnerships" scarcity note is good

### issues

1. **hero repeats homepage CTA copy verbatim** — "I take on a small number of partnerships at a time. If you've created something original and need a technical partner who builds with you for the long haul, I'd like to hear about it."
   - this exact copy appears on 3+ pages. for the contact page, be more direct:
   - **rewrite**: "Tell me what you're building. I'll tell you if I can help."

2. **success message says "hello@astralintegration.co"** — rebrand incomplete. also appears in sidebar email link.

3. **no phone/whatsapp option** — for practitioners in mazunte and the wellness world, whatsapp is the default communication channel. consider adding a whatsapp link as an alternative.

4. **sidebar note about selectivity** — "Not every inquiry becomes a project — but every inquiry gets a thoughtful response." — this is a good line but it's buried in a plain card at the bottom. consider promoting it to the hero area.

5. **form background uses bg-dark-card** — while other pages use bg-dark-bg for form inputs. minor inconsistency but noticeable if you look.

6. **no page-specific meta description** — the description references "artists, creatives, changemakers, practitioners, facilitators, and organizations" — the full 6-word list again.

### copy fixes
- hero: simplify to "Tell me what you're building."
- update all astralintegration.co references
- consider adding whatsapp contact option

# audit: /tools (ToolsHub)

## verdict: decent lead-gen page. some tone issues vs rest of site.

### what works
- "no sign-up required, free forever, your data stays yours" — strong trust signals
- grid layout for tools is clean
- "why free?" section explains the business logic honestly
- separate ToolsNavigation/ToolsFooter creates a contained experience

### issues

1. **"Soul-Led Businesses"** — this phrase appears in the hero ("Tools for Soul-Led Businesses") and the brand context of the content creator tool. it's not used anywhere else on the site. the rest of the site says "artists, creatives, changemakers, practitioners, facilitators, and organizations" or variations. "soul-led" is a different register — more woo, less technical.
   - **rewrite hero**: "Tools for Independent Practices" or "Free Tools for Your Practice"

2. **emojis in the "why free" grid** — uses lock, lightning, target emojis. the rest of the site is emoji-free. these should be replaced with the accent-colored icons or simple glyphs used elsewhere.

3. **"Built by the team behind Astral Studio"** — jordi is the team. this reads like there's a larger organization. 
   - **rewrite**: "Built by Astral Studio"

4. **"We believe in giving first"** — AI slop territory. vague, generic.
   - **rewrite**: "These tools started as solutions we built for clients. We decided to share them."

5. **CTA "Book a Free Strategy Call"** with arrow and scale hover effect — more aggressive than any other CTA on the site. tone it down to match.

6. **no connection back to main site nav** — uses ToolsNavigation which might not have the same links. tools users should be able to navigate to /work, /pricing etc.

### copy fixes
- "Soul-Led Businesses" → "Independent Practices"
- remove emojis from why-free grid
- "Built by the team" → "Built by Astral Studio"
- tone down CTA

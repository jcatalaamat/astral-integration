# audit: /organizations (OrganizationsPage)

## verdict: ambitious page. good architecture thinking. some copy issues.

### what works
- "your methodology scaled. your infrastructure didn't." — excellent hero
- scale metrics (1000+ facilitators, 10000+ students, 15+ languages, 50+ countries) establish credibility
- problem/solution pairs are detailed and specific
- "why not hire a dev team?" objection with cost/time specifics ($200K, 6-12 months) is persuasive
- prototype section linking to /access is smart lead-gen
- organizations button text "Start the Conversation" (vs generic "Send Message") is well differentiated

### issues

1. **the scale metrics are generic** — "1,000+ certified facilitators" and "10,000+ students worldwide" — these sound specific but the body text admits they're hypothetical: "these aren't hypothetical numbers" then immediately references "methodologies LIKE Access Consciousness." so which is it? if these are real numbers from a specific org, name it. if they're estimates for the category, say so honestly.
   - **rewrite**: "These are the numbers we see across the organizations in this space. Your scale may be different — the architecture adapts."

2. **pricing inconsistency** — meta says "15% rev share, Systems tier" and the pricing card correctly shows "15%". good. this is the only audience page that matches.

3. **"The Honest Question" section** — same pattern.
   - **fix**: "The Question Every Director Asks"

4. **access consciousness name-drop** — line 158 mentions "Access Consciousness, breathwork lineages, and certification-based healing modalities." this is fine contextually but be aware: if access consciousness is a client or potential client, naming them as a generic example (rather than a case study) might feel presumptuous.

5. **prototype CTA "View Prototype"** links to /access — but the access page is a massive interactive demo, not a simple prototype page. the user might expect a quick preview, not a full app. consider adding a note: "interactive demo — takes 2 minutes to explore"

6. **og url: astralintegration.co** — rebrand

### copy fixes
- scale metrics: add context clarifying these are category-level numbers
- prototype CTA: add "interactive demo" qualifier
